Dungeons and Dragons Beginners Session

  • Start time: 04:00pm - End time: 11:00pm
  • Date: 03/06/2022
  • Price: $10.00
  • 10 out of 20 Spaces Available
  • Location: 44 Fickle Ave, Woolloongabba, QLD

Requirements

Please bring the following:
  • - Note taking book
  • - Dice
  • - A character or character idea (and character sheet if you have it)
  • - A role playing attitude :)

Additional Information

If you've been looking to get into Dungeons and Dragons but don't know where to start, look no further! We'll spend the first hour going over how to create a character if you don't know how to already, if you do you're more than welcome to come a bit later. After that we have some pretty tame missions lined up for everyone, we'll split into groups of about 6-8 so it isn't too overwhelming. If this sounds like something you'd be interested in, book a ticket and come along.
